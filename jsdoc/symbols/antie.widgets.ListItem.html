<private> antie.widgets.ListItem(id)
The ListItem widget is a container widget that is used by the antie.widgets.List widget when set to List.RENDER_MODE_LIST. If you wish to control the classNames and id of list items, you can manually create them in your component/formatter and append them to the list. Otherwise, they will be automatically generated and will wrap other widgets you add to any antie.widgets.List widget when set to List.RENDER_MODE_LIST.
Parameters:
{String} id Optional
The unique ID of the widget. If excluded, a temporary internal ID will be used (but not included in any output).
