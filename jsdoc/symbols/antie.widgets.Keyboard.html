antie.widgets.Keyboard(id, cols, rows, keys)
On-screen keyboard widget.
Parameters:
{String} id Optional
The unique ID of the widget. If excluded, a temporary internal ID will be used (but not included in any output).
{Integer} cols
The number of columns of keys on the keyboard.
{Integer} rows
The number of rows of keys on the keyboard.
{String} keys
A string of characters which make up the keys, starting top-left and working row-by-row to the bottom-right. Special characters include: "-" DEL " " SPACE "_"
