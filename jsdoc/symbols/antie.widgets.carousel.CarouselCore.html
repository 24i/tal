Method Detail
alignNext(options)
Aligns the carousel to the next enabled widget after that currently aligned. If no alignment has been performed previously it will align to the next enabled widget after that at index 0 If a wrapping strip and navigator are used the alignment will wrap to the start after the last widget is reached. If an alignment is in progress, the new alignment will be queued to start after the current alignment completes.
Parameters:
{Object} options Optional
An animation options object
{Number} options.fps Optional
The frames per second of the alignment, if using styletopleft animation
{Number} options.duration Optional
The duration of the alignment in ms
{String} options.easing Optional
The alignment easing function
{Boolean} options.skipAnim Optional
If set true, the alignment will complete instantly then fire any provided callback
{Function} options.onComplete Optional
A function which will be executed on completion of the alignment animation.
alignPrevious(options)
Aligns the carousel to the enabled widget before that currently aligned. autoCalculate(on)
Some widget strips peform calculations which require elements to be present in the document. By default these calculations are performed whenever their values might be invalidated (after appending elements for instance) This method can be used to be disable/enable this behaviour for performance optimisation.
